# AdvoDesk Database Scripts

This folder contains SQL scripts for the AdvoDesk Legal Case Management System database.

## Files

### 1. `schema_and_data.sql`
**Clean, documented SQL script with:**
- Complete database schema (all 7 tables)
- Sample data for testing
- Well-organized and commented

**Use this file when:**
- Setting up a fresh database
- Understanding the database structure
- Resetting the database to initial state

### 2. `advo_desk_complete_backup.sql`
**Full database dump including:**
- All tables with exact structure
- All current data
- Auto-generated by mysqldump

**Use this file when:**
- Creating a complete backup
- Migrating to another server
- Restoring exact database state

## How to Use

### Fresh Installation

```bash
# Create and populate database
mysql -u root -p < schema_and_data.sql
```

### Restore from Backup

```bash
# Drop existing database (WARNING: This deletes all data!)
mysql -u root -p -e "DROP DATABASE IF EXISTS advo_desk;"

# Restore from backup
mysql -u root -p < advo_desk_complete_backup.sql
```

### Reset Database

```bash
# Drop and recreate
mysql -u root -p -e "DROP DATABASE IF EXISTS advo_desk;"
mysql -u root -p < schema_and_data.sql
```

## Database Structure

### Tables

1. **admins** - Administrator users
2. **advocates** - Lawyer/advocate users with verification workflow
3. **users** - Base user table for foreign key references
4. **clients** - Client information
5. **cases** - Legal case records
6. **hearings** - Court hearing schedules
7. **documents** - Case-related documents
